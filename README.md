### simple Crud


### Installation process

### Step one
```sh
$ git clone https://github.com/RezaulRaaz/simple-crud.git
$ cd simple-crud
$ npm install
```

### Step Two

```sh
Edit .env.example file to .env
$ npm run dev
```

### Step Three

```sh
$ php artisan serve
Go to http://127.0.0.1:8000
```

### Step Four

```sh
Go to http://127.0.0.1:8000/register
create account
```
